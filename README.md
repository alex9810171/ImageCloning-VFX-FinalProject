# VFX final project: Image Cloning

## Environment
 - flask
 - gunicorn

## Instruction to run the code
1. Run the program
```
    $ .\RUN.cmd   (for Windows system)
    $ ./RUN.sh    (for Linux, Ubuntu system)
```

2. Open a web tab and browse <http://localhost:9000> to enjoy our Image Cloning system~
